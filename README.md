# Jovia Platform

A full-stack influencer marketing and e-commerce platform built with React (Vite) and Django.

## Features

- Multi-role system (Admin, Brand, Influencer, User)
- Product management
- Event management
- Chat system
- Booking management
- Complaint system

## Tech Stack

### Frontend
- React
- Vite
- CSS Modules
- MUI

### Backend
- Django
- SQLite

## Run Frontend

```bash
cd Client
npm install
npm run dev
