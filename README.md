# Jovia Full Stack Project

A full stack influencer marketing platform built using React.js and Django.

## Features

- User authentication
- Influencer registration
- Brand registration
- Product management
- Event management
- Chat system
- Booking management
- Complaint system
- Admin dashboard

## Tech Stack

### Frontend
- React.js
- CSS Modules
- Vite

### Backend
- Django
- SQLite

## Project Structure

```bash
Frontend/
Backend/
```

## How to Run

### Frontend

```bash
cd Frontend
npm install
npm run dev
```

### Backend

```bash
cd Backend
python manage.py runserver
```
